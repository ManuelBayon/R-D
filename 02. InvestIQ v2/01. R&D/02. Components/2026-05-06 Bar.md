# Responsabilités 

- `Dataclass` représentant une aggregation de ticks en basse fréquence (minutes, heures, jours, semaines).
- La responsabilité de la cohérence de la période et de l'instrument incombe aux composants effectuant l'aggregation ou l'utilisant.
# Attributs

- `timestamp: datetime`: timestamp d'ouverture de la bougie.
- `open/high/low/close: float`
- `volume: int`
# Invariants

- Timestamp:
	- Invariants: doit être `UTC`.

- OHCL:
	- Valeurs finies 
	- Valeurs non négatives
	- high >= OLC
	- low <= OHC

- Volume: 
	- Valeur entière
	- Valeur finie
	- Valeur non négative

# Schéma de données

2026-05-06 :  

```python
@dataclass(frozen=True)  
class Bar:  
    timestamp_utc: datetime  
    open: float  
    high: float  
    low: float  
    close: float  
    volume: int = 0  
```
