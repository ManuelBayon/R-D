## Stratégies de dispatch

### 1. Match / Case

```python
match event:
    case BarEvent():
        ...
```

✔ simple  
❌ implicite, difficile à auditer à grande échelle

---
### 2. Polymorphisme

```python
event.handle(engine)
```

✔ encapsulation  
❌ mélange data + logique  
❌ moins adapté si tu veux sérialiser / distribuer

---
### 3. Registry (dispatch table)

```python
handler = handlers[type(event)]
result = handler(event)
```

✔ explicite  
✔ testable  
✔ proche des systèmes prod
