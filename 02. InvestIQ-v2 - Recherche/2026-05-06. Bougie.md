`Bar` représente timestamp + OHLCV finalisée issue d’une agrégation de ticks.

Garanties : 
- Timestamp format datetime et timezone UTC.
- Valeurs (OHLC) finies non négatives et (high >= OLC et low  <= OHC)
- Volume entier fini et non négatif.

Si une des garanties est violée lève `InvalidBar`.

```python
@dataclass(frozen=True)  
class Bar:  
    timestamp_utc: datetime  
    open: float  
    high: float  
    low: float  
    close: float  
    volume: int = 0  
```
