# 2. Les 4 profils qui atteignent ces niveaux

---
## 1. System / Low-latency engineer (C++)

### Compétences

- C++ très solide (mémoire, cache, threading)  
- latence / performance (ns, µs)  
- réseaux (TCP, kernel bypass)  
- systèmes temps réel  
- debugging complexe

### Pourquoi payé

```
→ chaque microseconde = argent
```

### Compétences

### Pourquoi payé

```
→ construit les systèmes qui génèrent le PnL
```
